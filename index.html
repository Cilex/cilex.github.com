    • Cilex

    A command-line micro-framework for PHP, inspired by Silex

    Project maintained by Cilex Hosted on GitHub Pages — Theme by mattgraham

    Cilex aims to provide a lightweight framework for creating PHP CLI scripts. It is inspired by the Silex microframework for web applications.

    Just like Silex, Cilex also ties together various Symfony2 components as well as the Pimple Dependency Injection Container. 

    For those wondering, "Cilex" is pronounced "C-lex", while "Silex" is pronounced "S-i-lex".
